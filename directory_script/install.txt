1. Переходимо в обрану папку в командній строці
    cd назва_папки (без пробілів)
2. Активуємо віртуальне оточення
    source venv/bin/activate
3. Встановлюємо пакети, перелічені у requirements.txt:
    pip3 install -r requirements.txt
4. Перевіримо які пакети встановленні в віртуальному середовищі
    pip3 list
5. По завершенню роботи виконаємо деактивацію віртуального оточення
    deactivate
